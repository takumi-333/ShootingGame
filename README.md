
# シンプルインベーダー

JavaのSwingを利用したインベーダー風GUIアプリケーションを作成しました。
3日間ほどで制作したものですので、保守性等ツッコミどころは万歳ですがご容赦ください。

## Run Locally

eclipse上でこのリポジトリをクローンし、
eclipseで実行してください。
